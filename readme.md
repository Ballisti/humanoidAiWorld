# Humanoid AI World

A starter template for building and running the **Humanoid AI World** project locally.

## Prerequisites

- **Git** (latest)
- **Python** 3.10+ (if backend/AI services are used)


## 1) Clone the repository

```bash
git clone https://github.com/Ballisti/humanoidAiWorld.git
cd humanoidAiWorld
```

## 2) Install dependencies

``` terminal
pip install -r packages.txt
```

anthropic

numpy

python-dotenv

## 3) Configure environment variables

create .env

Set required values in `.env`

``` .env
ANTHROPIC_API_KEY="your_api_key"
```

## 4) Run main.py
